高中篇1

发表于 2023-09-25 分类于 正文-高中篇

“Tomori，起床啦，待会有个很重要的通知”
唔……
我在被子里咕隆着，发出不耐烦的声音。高中开学的第一天，我却完全没有起床的动力。
刚刚迷迷糊糊发了line后信息栏还是显示未读状态，听说祥子现在还没好起来，听说好多同学都因为这一轮爆发感染了。
crychic的活动，不对，学校！
好不容易考上了羽丘，总不至于第一天就因为不愿来学校闹出很大的笑话了吧。
祥子！啊不对，她明明告诉我不能因为是在隔壁班就整天只粘着她。要交朋友……..
对！这次面对新的同学，总要改变初中一开始畏畏缩缩的初印象，要自信，连初中的你都能融入到集体之中，在高中也一样可以。
即便如此，折腾了一整子后，还是有点被困意压倒了。昨晚和crychic的大家在lins上聊到了很晚，不知道究竟是几点才睡着的。
“Tomori，学校已经全面停课了哦，说后续的安排很快就会发”
知道啦，欸？！

时间一分一秒地过去，被迫待在家里后，我实在无事可做，便来到了之前收集石头的架子面前。那些不知道从哪收集来五颜六色的石头被仔细地分门别类，用的完全小时候为了分类创造的一系列标签。
棕色的石头就叫“棕石”，方块状的被写上了“方块”（某种小孩子才会犯的拼写错误）。倒是鹅卵石，它有着很明确的名字。当时掌握了这个词汇后自己还高兴了好久，直到现在爸爸还会偶尔抱怨自己整天在他的耳边念叨着“这是不是鹅卵石”。
这个收集着各种各样东西的架子完全是爸爸的功劳，那时候自己哪有力气搭建这么大的万一。
或许是因为爸爸过于溺爱自己，稍微长大一点后，我就有点不太愿意收集这些东西了。一是因为干这些事的时候老是想起那个幼稚天真的自己，二是家里人都认为这个小架子是珍贵的回忆，没必要收起来或者扔掉。

soyo也发来信息了，她是在上课上了一半后被临时叫回家的。月之森的校门口一反常态地停满了本不应该开到这里的高级商务车，至让一向波澜不惊的她都吓了一跳。
从soyo发来的消息可以看见那些“全副武装”的保镖，他们身着知识点西装，带着厚厚的N4口罩。再加上那副墨镜，几乎没有法能看清他们的脸。
练习怎么办，我把心里想的东西发了过去，很快信息就显示为了已读。
“不知道”
当事情完全超出掌控的时候，soyo反而会回答得十分诚实。她在慌张时才会卸下平时那种乖乖女的伪装，虽然没有明面上戳破，但相处了一年的大家早就已经对彼此知根知底了。
“哈？”
立希好像也起来了。
“发生什么事了？”
“你又没去学校吗？这可是开学的第一天，难怪考不上羽丘”
回话的自然是毒舌版的soyo。
随便闲聊了几句之后，大家好像就没什么话题了。

两小时后，见祥子还没有回应，我开始有点担心起来，便在群里问了muzimi能不能试着直接联系她。
“去不了，父亲他已经禁止我们随便出门了”
反应这么迅速，不愧是睦的爸爸？
见大家没什么办法，我还是放弃了，再次扑向床的怀抱。
这一觉我少有地睡到了两点钟才起来，见祥子在群里报了平安，才放下了心来。
没事就好。

被关在家里已经过去了一周，网课也陆续开设了。各个群聊里面也开始热闹起来了。
我是高松灯，请多指教。
小心翼翼地斟酌了好久用语，我还是只打出了这简短的一句话。
“是高松灯吗？我是xxxx，请多指教”
“初中时我看过你的live哦，很可爱啊tomorin”
“tomorin？什么时候”
“那我呢？”

一眨眼的功夫，我就几乎没有办法浏览完班级群里的信息了。由于群聊的存在，大家交流话题的速度比平时快了许多倍，而且人也多了起来，一对一的交流几乎没法持续多长时间。
我的打字速度比较慢，而没当我输入时，显示栏上面那个大大的typing总是历历在目，让人非常焦急。
“啊，tomorin打错字了，真是可爱呢”
“灯会很烦恼的，别老是欺负她”
班上的大家还是很友善呢，我小心翼翼地把“不好意思，打错字了”发送出去，然后扑在床上长呼了一口气。
社交，好可怕。
我这么想着

劳累的一天总算结束后，我打开了crychic的群聊，这短短的几天我们就已经互相发送了比过去一年之和还多的信息，想想也挺吓人的。
祥子的症状好转一些了，真怕什么时候会轮到我呢。

“大家一起讨论一下练习的事吧”
不知道被关在家里的哪一天，立希突然在群里说道。大家陆续感染了一次之后，现在应该不会出什么岔子了。
“在线合奏，做的到吗？”
muzimi的问题确实戳中了要领。
“很简单，你看这个”
立希分享来了一个网红乐团跨洋合奏的现场和练习视频，我惊讶于她怎么能找到这种资源。
“被关在家里那么多天，只能这样了”
不仅是立希，soyo好像也在这方面突飞猛进了。前不久她还问我为什么ip属地突然变了，我问了后才知道是父亲整了什么动态ip，我也不太明白他说的东西。

就这样磕磕碰碰地，我们开始了线上练习，没想到muzimi对那些软件的上手速度比我还慢，看来她才是真正意义上的大小姐。
祥子倒是没什么差错，病好了之后，她又恢复了往日的沉稳优雅，并且时不时会关心我们遇到的各种各样的问题。
祥子还是那么可靠真的是太好了。

大纲草稿

发表于 2023-09-25 分类于 哲学

最早的两篇博文算是给整个文章定主基调用到，而这篇则是为已经有了整体思路的作品重新做一个简介与梗概。
看了这篇文章后，我的脑中就回想起了一个看少歌时想法，那就是为什么不让mygo乐队，甚至是包括少歌99组面对这个我们之前都遇见过，但很少把它正视为描写因素的重大变故呢？
为此我准备将会以高松灯为第一视角重写mygo的故事，将时间线由初三加高一扩展至初二到大一。crychic将会是初二组建起来的乐队，而mygo的组建则要推迟到大一。
整个规划大约分为两大部分，part1是高松灯从初二到大一前的人生历程，目前已经完成了初二初三的crychic简要剧情，初升高的部分则尚未完成，但大概有了雏形。而part2将集合高松灯在大一期间发生的各种小故事，包括她在文学社团内发表的各种小文章，成员之间发生的各种故事，mygo的组建等等。
组建crychic时候的高松灯是一个如爱音一样双商爆表的高手，几乎避免了所有的坑并且成功组建了阳光向上的完全体的crychic。除此之外，crychic的五人被设定为类似于afterglow的同校好友，乐队已经顺风顺水地组建起来且发展到了一定规模，但潜在积累的隐患最终随着重大变故的到来在高中完全爆炸，crychic因为各种因素将变得比原作裂地还狠，完全没有复活的可能。大祥老师大概也会经历战吧假药中的内容（也许也不会，主要是根本原因太爆了）。
最好笑的是这个这个重大变故放在剧本里可能会让人骂娘，但它真的发生了，而且不应该忽视或者忘记它。
如果看完初二初三的crychic，那么这段故事将会是一段十分经典的青春乐团故事，甚至给人一种乌托邦的错觉。
然而代价是初三升高一的突然变故会显得非常突兀，怎么埋伏笔我还没想好。
灯是如何从爱音这种性格变化为原著中的性格也是重点所在，也就是高中篇的主要内容，所以我才决定用第一人称+纪录视角重现整个初二到高三间的内心活动，让这个巨大变化显得相对合理一些。虽然有很大的文学和夸张成分，但笔者多少有过一些亲生体验，还是有信心能够写好的。

文学部某刊物收录一 草稿

发表于 2023-09-24 更新于 2023-09-25 分类于 哲学

你不觉得收集石头是一种浪漫的爱好吗，我对素世如是说道。我对地理的喜爱可能得追溯到小时候看各种科普纪录片的时候，似乎映照了康德的那句话，对于星空和脚下的这颗星球，随着长大我的心中就只剩下无限的敬畏与向往了。
真的吗？奈奈提到了她的一个同窗，他撕下了那颗用于送人的矿石上面的标签，后果可想而知。
虽然说素世可能完全误解了我所指的浪漫，却不禁令我想起了自己为什么向往星空和自然，却从未真正收集过石头或者进行任何天文部的理由。
初中时班上有一个真正热爱天文地理的女孩子，能对全班同学将恒星之间的星等、亮度之类侃侃而谈的她与我这种止步于表面的人完全不一样。
可惜，她是所谓校园暴力和孤立的受害者，那次比其他同学精心了不知多少倍的展示以灾难版的指责和挑刺而收尾，她不是日菜，没有那种即便面对千夫所指也毫无顾忌地展示自己才华的底气。彼时的我、貌似什么也没意识到，只是和朋友用电脑中的教科书图片糊弄完了老师，一般情况下其他同学对这种展示是头也不抬的。
当我某一天突然回忆起这件事的时候，才发现自己的房间根本没有为这种爱好提供一席之地，取而代之的是撕了又贴的海报，买来以后就不了了之的乐器。
真是虚伪呢，我一边抱怨着自己一边前往了科学馆。上了大学后我就再也没有碰过物理这种基础学科，反倒和国小生一起津津有味的在领队姐姐的带领下围着那堆分不清门类的标本和星图转圈。
当我觉得自己不再能够维持国小生那种持续的活力之后，一直无聊地用手机打发时间的爱音恰到好处地给了我一个台阶下。在那之后我们又沿着台场的海滨公园走了几圈，然后我们这次短暂的重逢就结束了，我第二天就要去京都，而她的假期也马上就要结束了。
那次东京之行后我开始试着真正拾取这个爱好，结果还是在天文部的门口迟迟不敢前进。
在日菜的建议下，我加入了文学部，除了校刊外文学部还有很多刊物需要写手，也自然有一个适合我的的职位，所以就把这篇文章作为自己收录入这份刊物的第一篇。
文学部的期刊有很多，最后也不知道除了我之外能有几个人能看到这篇充满了负能量的小文章呢(笑)，毕竟自己也不是为了完整性和类似与前进的这种主基调而强行给文章结尾的人。

素睦假药

发表于 2023-09-03 更新于 2023-09-04 分类于 搬运

这个睦素是作者在第12集播出之前写的，看完第12集之后作者甜甜地笑了：是的就是这样，，看似冷漠的soyo才是自觉被辜负而放不下的那一个

太好吃了请大家都来吃！

*出于个人喜好使用了“素世”这个译名；包含了一点睦素祥△要素

↓

素世玩笑似的提出交往。睦拎着那只铜色的小水壶浇花，听素世讲今天遇到了什么有趣的事，一直讲到新乐队的进展如何。素世说话，语调婉转而柔和，尾音上扬，与花园中鸟儿的鸣叫同调。睦偶尔做些回应，大部分时候只是安静地听。素世讲完，才发现睦不再盯着她格外关照的小黄瓜，视线不知何时转移到了自己身上。原来那只水壶里的水早已经用完了。素世心中微动，问道：“小睦喜欢我吗？”

睦的眼睛微微睁大了，人偶般精致而静止的面庞终于出现了神情的变化。

“啊……我说了奇怪的话呢。我们回教室吧。”

“喜欢。”

“什么？”

睦抱着空空的水壶，低着头回答：“喜欢。”

素世其实并不意外，但还是愣住了。她以为，睦不会回答这种棘手的问题。就像她问及祥子时，睦往往以沉默作答。真的听到睦说喜欢，竟觉得有些感动。

“我也喜欢小睦。”她放缓了声音。

睦眨了眨眼，唇角上扬。

很淡很淡的笑容，就像假面上的一丝裂缝。

看到别人笑时，自己也会想露出笑容。更何况她是素世，待人春风和煦的素世。素世知道自己也笑了。至此，话题就可以安然结束了。

她理应感到高兴的。然而，杂乱的思绪如不和谐音掠过，一口小钟在她胸膛里不轻不重地敲了一下。她听到自己用平时聊天那样轻快的声音说：“那，要不要和我交往？”

昙花一现的笑容敛去了，睦眉毛撇下来，就这样看着素世。这副表情，许久以前，一起坐在卡拉ok包厢里时也见到过。是在困惑吗？素世想。是啊，也不能太捉弄人了。她正要开口，睦却点了点头。

“咦～？没关系吗，睦？和女孩子一起交往也？”

“和素世一起……”睦把水壶放到水龙头底下，水花蹦出，声音不大，足够盖住她的低语。

长休的课间铃，悠扬地奏响。素世站在原地，半晌，走过去，回到教室走廊。睦跟在她身后，一如往常。

放学后，两个人出了教室，穿过人群，向同学道别，又来到花园。月之森的花园常有人修剪打理，饶是如此，仍芳草萋萋，枝蔓繁茂。素世坐在秋千上，睦倚着栏杆，看斜阳坠入湖水，赤色追逐风云，熏暖了素世蜷曲的长发。傍晚总是美得像另一个世界，而一切美丽中，素世的美最为惊心动魄。

“小睦知道交往意味着什么吗？”

“……”

素世笑了笑。她蔚蓝的双眸，比天空更像天空。

睦受了蛊惑般，把制服包搁在另一架秋千上，站定在素世身前，握住了她的手。

“是这样吗？”

透明的晚风吹起水手服的裙摆，浅灰领巾飘飘摇摇，将素世的视野遮蔽了彻底。被握住的手承载了夕晖的光热，她不无惊讶地抬起头。睦背对着陌生的傍晚的世界，还是那双熟悉的，不变的眼睛。

于是素世和睦开始交往。大概算是交往吧。日常发生的变化并不多，只是多了一些亲密的举止。素世更经常地黏着睦，会在别人看不到的角落，轻轻抱住睦。睦比她矮一些，抱在怀里十分合宜，只露出一个脑袋。顺滑的浅色发丝，摸上去惹人怜爱。睦还是安安静静浅浅淡淡的，但不知怎么的，素世知道她喜欢被这样对待。有一瞬间，她想：是不是不管什么行为，只要自己愿意，睦就不会拒绝？下一刻，祥子的身影伴着CRYCHIC的回忆浮上心头，顿时打碎了她的念想。近在眼前的睦，仿佛远在天边的祥子一样遥远。素世抱着睦，总是先尝到甜蜜的温暖，又泛起苦涩的寂寞。想到这里，便松开了手，而睦并不挽留。素世不明白睦。越是熟悉，越是不明白睦。睦说喜欢，究竟是怎样的喜欢，若是喜欢，又为什么从不做她希望的事情。是因为自己做的还不够多吧；是因为自己太手下留情吧。她或许是有些恨睦的。因此，说出“这都是睦的错”时，才感到痛快。

结果还是破镜难圆。素世从祥子家离开时，已经分不清耳鸣和蝉鸣。不可能有更想死的时候。最失魂落魄的模样，全被睦看在眼里。睦没有离素世而去，但也仅此而已。说实话，素世更希望睦离远点，并且把看到的东西都忘掉。

再怎么想死，只要还活着，就得上学。吹奏部的同窗们互道贵安，平稳的生活，让素世终于有逃避的地方。睦亦步亦趋，素世视若无睹。不想再流泪。不想再受挫。不想再费心。什么都不想。祥子能做到的无情，素世也要做到。维持冷漠，伤害他人，无一不需要精力。她实在累极了，靠一口气撑着，应付了来讨说法的立希。这时候就体会到了睦的省心。哪怕她把睦当空气无视，睦也没有凑上来起争执，而是默默地退回原地。但就是这点叫人生气。直到从灯那里听说睦去找过灯，素世才像醒来一样。

“睦说素世也迷路了。”

是吗。

“是吗？”

擅自说出那种话，那家伙又知道什么呢？

尽管她确实一直在看着。

被素世揪住衣领时，她也就这样看着。

——在你眼里，我究竟算什么？

那高高在上，八风不动的沉默，素世再也无法忍受。就着拽衣领的力，她把睦狠狠推倒在地，按下急促的呼吸，堪称轻柔地吻了上去。

露水滴落，鸟儿展平羽翼，飞入不再蔚蓝的天空。夏天过去了。
