大纲草稿

发表于 2023-09-25 分类于 哲学

最早的两篇博文算是给整个文章定主基调用到，而这篇则是为已经有了整体思路的作品重新做一个简介与梗概。
看了这篇文章后，我的脑中就回想起了一个看少歌时想法，那就是为什么不让mygo乐队，甚至是包括少歌99组面对这个我们之前都遇见过，但很少把它正视为描写因素的重大变故呢？
为此我准备将会以高松灯为第一视角重写mygo的故事，将时间线由初三加高一扩展至初二到大一。crychic将会是初二组建起来的乐队，而mygo的组建则要推迟到大一。
整个规划大约分为两大部分，part1是高松灯从初二到大一前的人生历程，目前已经完成了初二初三的crychic简要剧情，初升高的部分则尚未完成，但大概有了雏形。而part2将集合高松灯在大一期间发生的各种小故事，包括她在文学社团内发表的各种小文章，成员之间发生的各种故事，mygo的组建等等。
组建crychic时候的高松灯是一个如爱音一样双商爆表的高手，几乎避免了所有的坑并且成功组建了阳光向上的完全体的crychic。除此之外，crychic的五人被设定为类似于afterglow的同校好友，乐队已经顺风顺水地组建起来且发展到了一定规模，但潜在积累的隐患最终随着重大变故的到来在高中完全爆炸，crychic因为各种因素将变得比原作裂地还狠，完全没有复活的可能。大祥老师大概也会经历战吧假药中的内容（也许也不会，主要是根本原因太爆了）。
最好笑的是这个这个重大变故放在剧本里可能会让人骂娘，但它真的发生了，而且不应该忽视或者忘记它。
如果看完初二初三的crychic，那么这段故事将会是一段十分经典的青春乐团故事，甚至给人一种乌托邦的错觉。
然而代价是初三升高一的突然变故会显得非常突兀，怎么埋伏笔我还没想好。
灯是如何从爱音这种性格变化为原著中的性格也是重点所在，也就是高中篇的主要内容，所以我才决定用第一人称+纪录视角重现整个初二到高三间的内心活动，让这个巨大变化显得相对合理一些。虽然有很大的文学和夸张成分，但笔者多少有过一些亲生体验，还是有信心能够写好的。

回忆:高中篇(1)

发表于 2023-09-25 分类于 草稿

虽然进入了特优班，但我也没有自负到可以自诩新生代表的地步。
开学典礼之前，就有不少学生因为生病而无法出席，我内心中的阴暗一面也开始默默期待着新生代表会不会也因为中招而缺席。
可好景不长，一个粉色的身影还是站在了演讲台之上。她毫无疑问是一个日本人，但国中三年是在英国度过的，据说还有在传说中的王立戏剧学院交换的经历，这可是之前准备圣祥的时候连那些身为翘楚的学姐们都望尘莫及的学校。若非她的身材完全没有在英国待了三年的样子，我看你还真会把她与丸山彩弄混。
这位名叫千早爱音的女生自然和我同一个班，还好巧不巧地坐到了我的后桌。第一节课的自我介绍结束后，我自然没忍住上去搭了话。
她似乎总能找到一些与别人的共同话题，而且似乎总能对任何一个侃侃而谈。令我意外的是，对于舞台和剧本方面，她的兴趣很明显要比其他的要高出一截。
凭借这些，我大概能自诩自己在第一天就和她成为了很好的朋友，可与我不同的是，王立那边好像更侧重综合教育，与只准备过舞台创造科的我相比，她甚至是能够算作能身兼a班和b班的全才。
很自然的，我们也聊到了crychic的话题。她似乎挺关心乐队成员的问题，甚至之前就已经和素世认识了。根据立希的说法，似乎很快也，只不过封锁结束后大家见面时才告诉我。
或许是出于神明的恶作剧，我们这来到学校的几百号人乘着开学的第一天认识了彼此，然后迎来了对过去一切日常生活的审判。

回忆(9)春假

发表于 2023-09-25 分类于 草稿

春天是辞旧迎新的季节，而我们也用春日影作为crychic在初中的最后一首曲子。国中到高中的假期其实并不漫长，重新开始的乐队活动很快就把大家好不容易凑出来的日程表挤地满满当当。
在为数不多的休息时光中，我们去巨蛋看来afterglow和rosilia的对邦，即使crychic的影响力在同龄乐团中已经十分夸张，但距离西武这种大场所还离得很远。
可惜的是，这次live的出席率并没那么高，不过那是因为一些不可抗力，大家也没有那么在意。
立希在与afterglow前辈们的交流中显得异常兴奋，而我也惊讶于自己的乐团活动竟然一直被她们看在眼里。
当时我的精力都放在与友希奈咨询更加专业的事情，反倒忽视了美竹兰对我们非常在意的事情。
这是素世不久前才告诉我的，当时美竹兰有找到她，因为她说afterglow在升入大学前也遇到了许多和我们类似的问题，并且十分庆幸我们好像已经避免了这些问题，不过她还是为此表示十分担忧。不幸的是，这份担忧很快就成为了事实，而且可能远比前辈们设想的要严重。
假期之余还有一些令人糟心的事情，比如说海对岸某国已经因为某新型流行病采取了最终的紧急措施，虽然国内还没有什么影响，但生活中越来越多的口罩还是不仅令许多人担忧起来。
在最早开学的素世进入月之森的一周前，我们去斯特拉斯堡做了一段短暂的旅行。这是国际钢琴大赛的举办地，祥子与此地颇有渊源。如今她重新开始了钢琴活动，并且把一堆比赛提上了日程。
随着素世她们的开学，假期也很快迎来了尾声。只可惜从斯特拉斯堡回来后祥子就感染了，所幸的是，我们还在一个学校，有的是机会能聚在一起。
时间终于迎来了开学的时候，而我们人生的齿轮将迎来最大的改变。

回忆:中学与cryschic与毕业典礼

发表于 2023-09-25 分类于 草稿

距离毕业典礼还有三天时间，crychic的活动也第一次重新开始了。与只负责主唱的我不同，要演奏乐器的其他人或多或少都出现了手法生疏的情况。睦保留的最好，但祥子却意想不到地出现了极大的退步，以至于我们不得不临时修改给新曲的谱子。
这种危机感反倒给了我们已经非常疲惫的内心一丝冲劲。在校内进行了三天的紧急特训后，我们最终给了毕业典礼最完美的谢幕。
可让我不解的是，自己没能在一切结束后和其他人一样哭出来。情感丰富的睦是第一个忍不住的，随后是立希，还有祥子，甚至一直习惯和我们斗嘴的素世都在微笑的同时忍不住落下了泪水。
这不是一切的结束啊，我们才刚刚开始啊，我微笑地安慰着她们。
那时候我还没想到，这不是开始的结束，而是结束的开始。

回忆:中学与crychic(6)

发表于 2023-09-25 分类于 草稿

我终究还是做出了权衡各项利弊后的最理性方案，那就是选择风险小且回报高的特优班考试。那段为了舞台创造科所做出的种种努力宛若一场短暂的梦境，让我差点踏入那条充满了未知但又瑰丽的道路。
之后的半个月过的非常快，只记得考试结束后我们一同去了北海道，积雪中的温泉就像暴风雨中的避风港，给我们疲惫且忐忑不安的内心一个临时的安息之所。
不知道是谁在我们都昏昏欲睡的时候嘟哝了一句“要组一辈子乐队”，但我猜大家应该都记住了这句话。
考试的结果很快出来了，好消息是素世，我还有睦都得到了最好的结果，但立希和祥子的结果却不太好。祥子没能进入特优班，虽然还是能拿到奖学金。而立希因为总是翘课更是一塌糊涂，如今的她只能尝试花咲川的补录考。
事到如今大家也顾不上中断了一个多月的乐队活动了，祥子要花些精力从其他方面补充自己的简历。最为关键的还是大家开始了对立希的特训，她整天翘课落下来的知识太多了，随便从街上拉一只野猫估计都比她聪明。
终于，赶在毕业典礼到来之前重新让所有人走上了正轨。
直到这时，睦才提起了我们已经小半年没有进才会开始珍惜的东西了吧。行乐队活动了。或许正如她所说，生活的重压真能让人忘记很多事情，日常可能就是那些失去了

回忆:中学与crychic(6)

发表于 2023-09-25 分类于 草稿

或许是由于学力相近的缘故，祥子、素世和我是一起学习的，对于大家的长项大家还是知根知底的。素世总是抱怨我的议论文总是会在严谨的论证结构中插入各种夸张但有力的论据，这无疑说是一种鬼才的表达方式，但总会让文章显得有点四不像。
这也让我开始担心起圣翔那边的考试，对于羽丘的应试来说，这种夸张的论述方式无疑是左右逢源把采分点都蹭了一遍。但是在创造科考试主观程度较高的剧本编写中，反倒多了非常多的不确定性。
另一个担忧的地方就是语言的含蓄与否，每次办完live后，我都会收集观众对于歌词的评价。虽然大多数评论都会指出我所创作的歌词都是“直击人心”，可一旦结合由此联想到的事物后，又会发现她们体会到的感情都完全不同。
比如说一位名叫Rikki的用户，她在听完春日影后就在SNS上写了一长串自己苦苦追求一个女孩子无果最后甘愿做注视她的骑士的发癫小作文，并且表示“啊，这唱的简直就是我”。
这个发癫小作文被祥子看到后，她确表示这是一个女孩一不小心成为了他人的神明这件事，然后询问起我的想法。
就像自己的文章被拿去出语文题的作者一样，我当然是毫无头绪的，毕竟我也不记得自己在创作春日影之前都看了哪些书。那时自己的思路远没有之后找到方法论和步骤后那么清晰。
转眼间距离考试就只有一周了，我在下定决心前最后一次向祥子咨询了意见。可这次的她却给出了不同于以往的模糊回答，她让我凭借自己的意志决定。

回忆:中学与crychic(5)

发表于 2023-09-25 分类于 草稿

考试季开始前crychic又举办了几次live，虽然产出和频率都有所下降，但是新创作的几首曲子不论是在质量上还是反响都不亚于之前的几个，但我总感觉自己在创作这些曲子的时候好像失去了什么。回看过去创作的歌词，总感觉它们只是由词语堆砌而成的建筑，而我丝毫无法从中解读出任何意义。有时候我不禁会想这些歌词是否过于直白，以至于任何受过基础教育的人都能理解其洋溢出来的充沛情感。
或许这是因为准备圣翔的考试时候接触许多关于舞台制作以及剧本编写的理论。比如说将戏剧和面具与一般的音乐演出结合起来。就像演员一样，面具成为了沟通表演者和观众的桥梁。借由虚假的角色之口，一些平时中无法说出的句子反倒可以直接传达出来。去圣翔参观的时候，示范的三年级学姐在表演塞尔瓦多雷时候的肺腑之言就一直笼罩在我的脑中，以至于我根本分不清那句话是对剧本中的远航者还是她的友人所说。
当我和祥子分享我对感想之时，她只是若有所思地点了点头。虽然作为大小姐的她完全是用相反的方式使用那副面具，但如果完全融入这个角色，比如说扮演月之森的学生，也是有趣的选择呢。她在说到这里的时候不由得戏虐地笑了一下，骰子已经掷下，选择了从羽丘升学道路的她又有何理由临时换道呢。
充实而紧张的备考生活就这样持续到了各学校放出具体考试日期的时候，和其他重点学府一样，圣翔的考试也分为多个阶段，而舞台创造科的入学考就好巧不巧地和羽丘的特优班入学考撞到了一起。
要么不做，要么做绝，出于公平考虑，各学校有限的尖子生名额不能给日菜这种天才全部揽下。即便是我这种上不去下不来的人也该做出自己的抉择了。

回忆:中学与crychic(4)

发表于 2023-09-25 分类于 草稿

国中二年级升三年级的那段时光，我们又陆续演出了诗超伴梵音打等曲目。就当我准备创作第五首曲子，也就是迷路日々的时候，父亲一反常态地带着严肃的眼神找到了我。
升学顾问在分析之后提出了让我去就读圣翔音乐学院舞台创造科的建议，这对于对未来丝毫没有明确构想的我来说确实算是一个出乎意料的选择。
这并不是一个容易做出的决定，但顾问却建议我可以先抱着试试看的态度先准备着。
我自然把这件事告诉了乐队里的其他人，同时想咨询一下她们的意见，毕竟圣翔的压力并非普通高中能对比的，这样下去高中后乐队也不可避免地将减少活动。与预想中的差不多，大家都认为这是一个很不错的选择，素世虽然在一开始对高中后可能更难聚在一起表示了些许遗憾，但还是为我感到高兴。
一如既往地借由日常的拌嘴安慰着素世，我把目光投向了心里最关心的祥子。从她的眼中虽然能看出一丝沮丧，但整体还是非常肯定且支持的，这让我的内心稍微有一些复杂。
乐队的未来随着升学的迫近开始笼罩上了一层阴影，与我一样，大家都需要考虑自己的未来。
素世最后还是决定以月之森为目标冲刺，因为她知道即使没有明说，母亲还是希望她在这所学校接触完全不同的圈子，在国中以及crychic，她已经度过了充实的中学生活了。睦的决定和素世一样，或者说，作为知名艺术家孩子的她和自己在一所学校本来就够奇怪的。祥子和立希决定报考羽丘，对于祥子来说，在以升学著称的羽丘拿奖学金比就读被成为大小姐学校的月之森更能证明自己一些。
或许正如新歌的名字一样，乐队的大家似乎都在迷茫中前进着。虽然未来依旧充满着不确定性，度过了这次小小危机的我还相信这只是对乐队一次微不足道的考验，谁知这是蝴蝶震动翅膀的开始。
